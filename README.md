## 📂 Featured Projects

### 📈 1. Indian Stock Market Prediction (LSTM Pipeline)
*(Located in `/indian-stock-prediction-lstm`)*

An advanced, out-of-core Deep Learning streaming pipeline designed to forecast equity movements using Long Short-Term Memory (LSTM) networks. 

**Key Architectural Highlights:**
* **Deep Learning Integration:** Upgraded the core predictive engine to an LSTM-based deep neural network, engineered to capture complex temporal dependencies in financial time-series data using custom 3D tensor sequence generation.
* **High-Performance Data Engineering:** Utilizes optimized storage formats (Apache Parquet and HDF5) and out-of-core data streaming to efficiently train on massive, larger-than-RAM datasets.
* **End-to-End Automated Pipeline:** Features a robust, modular ETL system for automated daily data ingestion from the NSE, feature engineering, model retraining and historical prediction archiving.
* **Actionable Intelligence:** The inference engine filters noise to generate and rank targeted "Top-K" daily trading candidates.
